# API-REST-Express-Mongo
API Rest de uma Livraria com Express e MongoDB - Curso da Alura

### Rotas:
##### GET
http://localhost:3000/autores
http://localhost:3000/autores/:id
http://localhost:3000/livros
http://localhost:3000/livros/busca
http://localhost:3000/livros/:id

##### POST
http://localhost:3000/livros
http://localhost:3000/autores

##### PUT
http://localhost:3000/livros/:id
http://localhost:3000/autores/:id

##### DELETE
http://localhost:3000/livros/:id
http://localhost:3000/autores/:id
